# rust-focus

A command-line productivity tool built with Rust (Todo + Pomodoro).

## Goals
- Build a practical CLI tool
- Support task management and a Pomodoro timer

## Planned Commands (Draft)
- `add <title>`: Add a task
- `list`: List all tasks
- `done <id>`: Mark a task as completed
- `delete <id>`: Delete a task
- `start <minutes>`: Start a Pomodoro timer
